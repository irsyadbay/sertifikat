#  Sertifikat & Pengalaman

Repositori ini berisi kumpulan sertifikat serta pengalaman saya di bidang cybersecurity dan teknologi informasi.

Fokus utama saya adalah pada keamanan aplikasi web dan riset kerentanan, khususnya dalam menemukan dan melaporkan celah keamanan secara etis.

---

##  Bidang yang Dipelajari
- Web Application Security
- Bug Hunting
- Vulnerability Research
- Basic Penetration Testing

---

##  Pengalaman
- Melakukan pembelajaran mandiri terkait keamanan aplikasi web  
- Menganalisis dan menguji potensi kerentanan pada sistem  
- Memahami konsep kerentanan
- Berkontribusi dalam pelaporan kerentanan secara bertanggung jawab

---

##  Tujuan
Terus mengembangkan kemampuan di bidang cybersecurity dan membangun karier sebagai:
- Penetration Tester
- Security Engineer
- Bug Hunter

---

##  Catatan
Sertifikat dan bukti pendukung tersedia dalam repositori ini sebagai bagian dari perjalanan belajar dan pengembangan skill saya.
